# PyPlyne Icon Concepts

Draft SVG concepts for the package/site icon.

Chosen direction: `pipe-token-blueprint.svg`, promoted to `../pyplyne-icon.svg`
and `../favicon.svg`.

- `pipe-loop.svg`: compact looped pipe, strongest favicon candidate.
- `pipe-elbow.svg`: literal pipe bend with a small flow arrow.
- `pipe-token.svg`: code-token inspired `|>` mark.
- `pipe-node.svg`: pipeline stages as connected nodes.
- `pipe-p.svg`: a pipe-shaped P mark.
- `pipe-ribbon.svg`: folded dual-pipe mark.

Preview locally at `/img/icon-concepts/preview.html` while the Docusaurus dev server is running.
